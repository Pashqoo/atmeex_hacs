# Логотип для интеграции

## Как добавить логотип

### Вариант 1: Добавить готовый файл

1. Получите логотип ATMEEX (PNG файл)
2. Сохраните его как `brand.png` в корне репозитория:
   ```
   /Users/pavelbakulin/bot/HA/atmeex_hacs_fixed/brand.png
   ```
3. Файл должен быть:
   - Формат: PNG
   - Размер: рекомендуется 512x512 пикселей или 256x256
   - Прозрачный фон (опционально)
   - Размер файла: до 200 KB

### Вариант 2: Скачать логотип с сайта

Если у вас есть URL логотипа ATMEEX, можно использовать его напрямую в `hacs.json`.

## Где используется

- **HACS**: Отображается в каталоге интеграций
- **Home Assistant UI**: Может использоваться в интерфейсе устройств
- **GitHub**: Отображается на странице репозитория

## Текущая конфигурация

Логотип настроен в:
- `hacs.json` - путь к изображению для HACS
- `manifest.json` - информация о бренде

После добавления файла `brand.png`, загрузите его в репозиторий.

## Проверка

После загрузки файла на GitHub:
1. Проверьте URL: `https://raw.githubusercontent.com/Pashqoo/atmeex_hacs/main/brand.png`
2. Убедитесь, что изображение отображается
3. В HACS логотип появится автоматически после обновления интеграции
